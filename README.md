# DptoLibre Web

Versión corregida para publicación rápida.

Esta versión usa la imagen completa de la propuesta visual para evitar el problema de recortes en `hero.jpg`, `admin.jpg`, `owner.jpg`, `stories.jpg` y `footercta.jpg`.

Archivos:
- index.html
- styles.css
- landing-propuesta.png

Para GitHub Pages:
1. Subir estos 3 archivos a la raíz del repositorio.
2. Hacer commit.
3. Esperar el deploy.

Para GoDaddy:
1. Subir estos 3 archivos dentro de `public_html`.
2. El `index.html` debe quedar directamente en `public_html/index.html`.
